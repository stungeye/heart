# Math Heart

Deployed to Netlify: [heart-math.netlify.app](https://heart-math.netlify.app/)

A Valentine's Day sketch built with [p5.js](https://p5js.org/) that draws and animates a heart using the parametric heart curve:

- `x = 16 sin³(t)`
- `y = 13cos(t) − 5cos(2t) − 2cos(3t) − cos(4t)`

The heart pulses with a double-sine heartbeat rhythm, surrounded by mini-heart particles that drift outward with each beat.

- `s(t) = 1 + 0.06 sin(ωt) + 0.03 sin(2ωt)`
