# Math Heart

A Valentine's Day sketch built with [p5.js](https://p5js.org/) that draws and animates a heart using the parametric heart curve:

- `x = 16 sin³(t)`
- `y = 13cos(t) − 5cos(2t) − 2cos(3t) − cos(4t)`

The heart pulses with a double-sine heartbeat rhythm, surrounded by mini-heart particles that drift outward with each beat. The math is displayed as an elegant overlay.

- `s(t) = 1 + 0.06 sin(ωt) + 0.03 sin(2ωt)`

## Usage

Open `index.html` in a browser.
